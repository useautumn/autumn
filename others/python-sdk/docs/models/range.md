# Range

Time range to aggregate events for. Either range or custom_range must be provided


## Values

| Name           | Value          |
| -------------- | -------------- |
| `TWENTY_FOURH` | 24h            |
| `SEVEND`       | 7d             |
| `THIRTYD`      | 30d            |
| `NINETYD`      | 90d            |
| `LAST_CYCLE`   | last_cycle     |
| `ONEBC`        | 1bc            |
| `THREEBC`      | 3bc            |
