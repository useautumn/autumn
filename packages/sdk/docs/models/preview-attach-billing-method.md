# PreviewAttachBillingMethod

'prepaid' for upfront payment (seats), 'usage_based' for pay-as-you-go.

## Example Usage

```typescript
import { PreviewAttachBillingMethod } from "@useautumn/sdk";

let value: PreviewAttachBillingMethod = "prepaid";
```
