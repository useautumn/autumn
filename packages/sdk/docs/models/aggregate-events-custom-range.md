# AggregateEventsCustomRange

Custom time range to aggregate events for. If provided, range must not be provided

## Example Usage

```typescript
import { AggregateEventsCustomRange } from "@useautumn/sdk";

let value: AggregateEventsCustomRange = {
  start: 1540.52,
  end: 8116.03,
};
```
