Autumn Favicon System

Environment-specific favicons to distinguish between dev, staging, and production

How It Works

  • Development: Orange favicon when NODE_ENV=development
  • Staging: Light purple favicon when VERCEL_ENV=preview
  • Production: Original purple favicon (default)

Switching Designs

To use an alternative design, edit /lib/get-favicon.ts and change the return value to your preferred file:

if (process.env.NODE_ENV === "development") {
    return "/favicon-dev-alt.svg"; // Use alternative orange
}
