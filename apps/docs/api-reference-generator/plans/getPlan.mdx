Retrieves a single plan by its ID. Returns the latest version by default.

### Common Use Cases

<CodeGroup>

```typescript Get a plan
const plan = await autumn.plans.get({
  planId: "pro_plan"
});
```

```typescript Get a specific version
const plan = await autumn.plans.get({
  planId: "pro_plan",
  version: 2
});
```

</CodeGroup>
