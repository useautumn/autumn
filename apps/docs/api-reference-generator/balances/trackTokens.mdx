<Note>
  Track AI token usage against a customer's AI credit system balance. Converts token counts to a dollar cost using OpenRouter pricing and your configured markup, then deducts from the customer's credit balance.
</Note>

### Common Use Cases

<CodeGroup>

```typescript Track AI token usage
await autumn.balances.trackTokens({
  customerId: "cus_123",
  modelId: "anthropic/claude-opus-4-6",
  inputTokens: 1000,
  outputTokens: 500
});
```

```typescript Track with explicit feature
await autumn.balances.trackTokens({
  customerId: "cus_123",
  featureId: "ai_credits",
  modelId: "anthropic/claude-haiku-4-5",
  inputTokens: 2000,
  outputTokens: 1000
});
```

```typescript Track with properties
await autumn.balances.trackTokens({
  customerId: "cus_123",
  modelId: "openai/gpt-4o",
  inputTokens: 500,
  outputTokens: 200,
  properties: {
    conversation_id: "conv_abc123",
    prompt_type: "summarization"
  }
});
```

</CodeGroup>
